dsgn / identidad visual

Diseño de identidad visual para DSGN by Resta - una plataforma creativa de espacios y objetos para el bien común, un universo de productos diseñados para crear historias.

En 2021, tuve el privilegio de llevar adelante el proceso de diseñar este proyecto, acompañada de Paola Resta y Marcela Terriles, co-fundadoras de DSGN. Actualmente, esta plataforma se encuentra en re-lanzamiento previsto para 2025.
